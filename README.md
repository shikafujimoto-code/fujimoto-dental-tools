# fujimoto-dental-tools

ふじもと歯科向けの院内業務・患者コミュニケーション支援ツールです。

## 主な画面

- `/` - 患者説明資料ジェネレーター
- `/instagram` - Instagram投稿下書きプロンプト生成プロトタイプ
- `/demo` - チャットウィジェットのデモ
- `/transcript` - チャットログ確認画面（Basic認証）
- `/blog` - SEOブログ作成支援画面（Basic認証）

## 開発

```sh
deno task dev
```

## 起動

```sh
deno task start
```

## Instagram投稿プロンプトプロトタイプ

`/instagram` では、医療法人ふじもと歯科のInstagram投稿下書き生成プロンプト設計書に沿って、以下をブラウザ上で確認できます。

1. 投稿カテゴリーとサブテーマの選択
2. お知らせ・採用投稿向けの具体情報入力
3. AIへ送る完成プロンプトの組み立て
4. 出力形式確認用のサンプル下書き生成
5. 表示内容のコピー

実際のAI API接続前に、院内で入力項目・出力フォーマット・医療広告ガイドライン確認フローをすり合わせるための静的プロトタイプです。
